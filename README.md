# 8-Puzzle Solver using A* Search

## Overview

Developed an intelligent solver for the classic 8-puzzle problem using the A* search algorithm in Common Lisp. The program automatically finds the optimal sequence of moves from an initial board configuration to the goal state by combining path cost and heuristic evaluation.

## Features

- Automatic 8-puzzle solving
- A* Search algorithm
- Heuristic-based path finding
- Optimal solution generation
- State expansion and node evaluation

## Technologies

- Common Lisp
- A* Search
- Artificial Intelligence Search Algorithms

## Learning Outcomes

- Implemented informed search algorithms
- Applied heuristic functions to optimize search efficiency
- Explored state-space search and graph traversal techniques
